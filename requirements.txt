### 💡 常见问题

**Q：`Arm_Lib` 从哪里获取？**  
A：`Arm_Lib` 是特定机械臂厂商提供的驱动库，通常随硬件附赠或从厂商 GitHub 下载。如果你的机械臂是其他品牌，需要替换为对应的驱动库，并调整 `write_arm_pose` 函数中的调用方式。

**Q：我在树莓派上运行报错怎么办？**  
A：树莓派建议使用 `opencv-python-headless`（无 GUI 依赖），并确保摄像头已启用：`sudo raspi-config` → Interface Options → Camera → Enable。

**Q：`ipywidgets` 在 Jupyter Lab 中不显示画面？**  
A：需要额外安装 Jupyter Lab 扩展：
```bash
jupyter labextension install @jupyter-widgets/jupyterlab-manager
```
