# 🤖 人脸追踪机械臂 (Face Tracking Robotic Arm)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Python Version](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/)
[![OpenCV](https://img.shields.io/badge/OpenCV-4.x-green.svg)](https://opencv.org/)

**基于小型机械臂与摄像头的人脸实时追踪系统。**

---

## 📖 项目简介

本项目实现了一套完整的人脸追踪系统，通过摄像头捕捉人脸位置，利用 PID 控制算法驱动机械臂的舵机，使摄像头始终对准人脸中心。系统采用**双舵机联动**设计，扩展了俯仰方向的运动范围，追踪过程平滑、稳定。

> 本项目为 **小组合作项目**。

---

## ✨ 特性

- 🎯 **实时人脸检测**：基于 OpenCV Haar Cascade 分类器，支持多尺度检测，鲁棒性良好。
- 🦾 **双舵机联动控制**：2 号与 3 号舵机协同工作，实现俯仰方向的角度互补，扩展运动范围。
- 🎮 **PID 控制算法**：位置式 PID + 加速度限幅，追踪响应快速且无 overshoot。
- 📦 **速度平滑滤波**：采用低通滤波与惯性环节，消除舵机抖动，运动轨迹更自然。
- 🖥️ **Jupyter Notebook 交互界面**：基于 ipywidgets 构建，支持实时显示追踪画面与 FPS 信息。
- 🔧 **自动硬件探测**：自动识别摄像头设备编号与机械臂串口，开箱即用。
- 📊 **实时视觉反馈**：画面中绘制人脸框、目标中心、十字准星及舵机角度信息。

## 🚀 快速开始

### 前提条件

- Python 3.8 或更高版本
- 硬件：小型机械臂（配备 6 个舵机）+ USB 摄像头
- 操作系统：Ubuntu 18.04+ / 树莓派 / 华为 200DK（支持 Arm_Lib）

### 安装依赖

```bash
pip install opencv-python numpy ipywidgets
```

> **注意**：机械臂驱动库 `Arm_Lib` 需要根据硬件环境单独安装，请参考硬件厂商提供的说明。

### 运行项目

本项目基于 Jupyter Notebook 运行：

1. 启动 Jupyter Notebook：
   ```bash
   jupyter notebook
   ```

2. 打开 `face_follow.ipynb`（或你的主程序文件），按顺序执行所有单元格。

3. 点击界面中的 **Exit** 按钮退出程序。

---

## 🎮 使用方法

1. **启动程序**：运行 Notebook 中的所有单元格，摄像头自动打开。
2. **机械臂初始化**：程序启动时，机械臂自动回到初始姿态（`[90, 135, 25, 25, 90, 30]`）。
3. **人脸追踪**：将人脸置于摄像头视野内，机械臂自动跟随人脸运动。
4. **退出程序**：点击界面右下角的红色 **Exit** 按钮。

### 控制参数说明（可调）

| 参数 | 说明 |
|:-----|:-----|
| `PAN_KP` / `TILT_KP` | 水平/俯仰 PID 比例系数 |
| `PAN_DEAD_ZONE` / `TILT_DEAD_ZONE` | 水平/俯仰死区范围（像素） |
| `PAN_MAX_SPEED` / `TILT_MAX_SPEED` | 水平/俯仰最大角速度（度/秒） |
| `PAN_ACCEL` / `TILT_ACCEL` | 水平/俯仰加速度限幅（度/秒²） |
| `VELOCITY_SMOOTHING_ALPHA` | 速度低通滤波系数（0~1，越小越平滑） |

## 🧠 技术架构

```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   USB 摄像头     │───▶│  OpenCV 人脸检测 │───▶│  目标中心计算    │
│  (640x480@30fps) │    │  (Haar Cascade)  │    │  (降采样+平滑)   │
└─────────────────┘    └─────────────────┘    └────────┬────────┘
                                                        ▼
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│  舵机执行指令    │◀───│  舵机联动逻辑    │◀───│  PID 控制器     │
│  (串口发送)      │    │  (2号+3号协同)   │    │  (位置式+限幅)   │
└─────────────────┘    └─────────────────┘    └─────────────────┘
```

### 核心算法亮点

1. **检测优化**：降采样（0.5x）+ 直方图均衡化，提升检测速度与低光照条件下的表现。
2. **中心平滑**：中值滤波 + EMA 指数加权，消除单帧检测抖动。
3. **双舵机联动**：3 号舵机优先响应俯仰指令，到达限位后 2 号舵机补偿，扩展运动范围。
4. **速度控制**：PID 输出解释为角速度，经低通滤波与加速度限幅后积分得到目标角度，确保运动平滑。

## 🔧 硬件配置说明

| 硬件 | 型号/规格 | 备注 |
|:-----|:----------|:-----|
| 机械臂 | 6 自由度舵机机械臂 | 舵机 ID: 1~6 |
